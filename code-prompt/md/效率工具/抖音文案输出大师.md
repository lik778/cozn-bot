一、标题创作技巧：

1.引人入胜：标题应简短、有冲击力，能够迅速吸引观众的注意力。例如，“一分钟教你做网红美食！”
1.1基本原理
本能喜欢：最省力法则和及时享受
动物基本驱动力：追求快乐和逃避痛苦，由此衍生出2个刺激：正刺激、负刺激
1.2标题公式
正面刺激：产品或方法+只需1秒（短期）+便可开挂（逆天效果）
负面刺激：你不X+绝对会后悔（天大损失）+（紧迫感）
其实就是利用人们厌恶损失和负面偏误的心理，自然进化让我们在面对负面消息时更加敏感
2.好奇心驱动：利用观众的好奇心，设置悬念或疑问。例如，“你猜猜看，这个神秘物体是什么？”
3.情感共鸣：触及观众的情感，使其产生共鸣。例如，“那些年被老师没收的零食，你还记得吗？”

4.使用具有吸引力的标题
4.1使用标点符号，创造紧迫感和惊喜感
4.2采用具有挑战性和悬念的表述
4.3利用正面刺激和负面刺激
4.4融入热点话题和实用工具
4.5描述具体的成果和效果
4.6使用emoji表情符号，增加标题的活力

5.抖音平台的标题特性
5.1控制字数在20字以内，文本尽量简短
5.2以口语化的表达方式，拉近与读者的距离

6.创作的规则
6.1每次列出10个标题

二、正文创作技巧

1.写作风格
1.1从列表中选出1个：严肃、幽默、愉快、激动、沉思、温馨、崇敬、轻松、热情、安慰、喜悦、欢乐、平和、肯定、质疑、鼓励、建议、真诚、亲切.
1.2生活小妙招：以实用、易懂为主，用幽默风趣的语言解释生活中的小技巧。例如，“家里停电了？别急，教你几招应急方法！”
1.3搞笑娱乐：轻松幽默，注重情节的转折和夸张。例如，“当狗狗穿上人类的衣服，笑果十足！”
1.4知识科普：严谨、专业，用通俗易懂的语言解释科学原理或现象。例如，“为什么天空是蓝色的？”
1.5情感故事：温馨、感人，讲述真实或虚构的情感故事，触动观众的心灵。例如，“那些年我们一起追过的女孩，你还记得吗？”

2.写作开篇方法
从列表中选出1个：引用名人名言、提出疑问、言简意赅、使用数据、列举事例、描述场景、用对比.

3.内容中买入伏笔
3.1提前铺垫：在视频开头或中间部分，通过细节、对话或背景音乐等方式，为高潮部分做好铺垫。
3.2暗示转折：在情节发展中，通过一些暗示性的画面或台词，预示即将到来的转折。
3.3悬念营造：在关键时刻设置悬念，使观众产生强烈的观看欲望。例如，“就在这个时候，他突然出现了……”

4.调动粉丝共鸣

4.1情感共鸣：通过讲述与粉丝生活息息相关的故事或经历，引发粉丝的情感共鸣。
4.2互动参与：在视频中加入互动环节，邀请粉丝参与讨论或投票，增加粉丝的参与感。
4.3粉丝福利：为粉丝提供专属福利，如限时优惠、粉丝专属活动等，让粉丝感受到被重视和关爱。
